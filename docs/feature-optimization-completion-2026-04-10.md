## ✅ 已完成的优化

### 1. 撤销删除功能

**文件：** `apps/web/components/inquiry/InquiryCartDrawer.tsx`, `apps/web/components/inquiry/InquiryItem.tsx`

**实现内容：**
- 添加了撤销状态管理 (`undoItem`)
- 添加了撤销计时器 (`undoTimerRef`)
- 添加了撤销 UI 组件：
  - 蓝色背景的撤销提示条
  - 显示"已删除 1 件商品"
  - "撤销"按钮可恢复商品
  - 5秒自动过期
- 更新了 InquiryItem 组件：
  - 支持 `onRemove` 回调
  - 如果提供回调则调用回调，否则直接删除
- 自动清除计时器防止内存泄漏

**用户体验提升：**
- 防止误删商品
- 5秒内可撤销
- 清晰的视觉反馈

---

### 2. 产品筛选 URL 状态记忆

**文件：** `apps/web/app/[locale]/products/page.tsx`

**实现内容：**
- 使用 `useSearchParams` 和 `useRouter` 管理 URL 参数
- 分类筛选参数：`?category=body-scales`
- 搜索关键词参数：`?search=scale`
- 完整的 URL 参数更新逻辑：
  - 分类变更立即更新 URL
  - 搜索输入防抖（300ms）后更新 URL
  - 页面加载时从 URL 恢复筛选状态
- 增强的筛选 UI：
  - 搜索框添加清除按钮
  - 显示当前筛选结果数量
  - "清除筛选"按钮一键重置
- 添加 `aria-label` 提升可访问性

**用户体验提升：**
- 筛选状态可分享（复制 URL 给其他人）
- 刷新页面不丢失筛选状态
- 浏览器前进/后退支持

### 3. 产品列表空状态引导

**文件：** `apps/web/app/[locale]/products/page.tsx`

**实现内容：**

  - "清除所有筛选"按钮（仅当有筛选条件时显示）

**用户体验提升：**
