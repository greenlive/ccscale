# B2B产品详情页 - 产品主图与详情图展示设计

**日期:** 2026-05-05
**状态:** 已批准
**版本:** 1.0

## 背景

参照阿里巴巴国际站产品详情页的布局设计，改进CC Scale B2B外贸平台的 产品详情页产品图片展示方式，使其更符合B2B买家的浏览习惯。

## 设计决策

### 1. 图片存储结构

| 字段 | 类型 | 数量 | 说明 |
|------|------|------|------|
| `mainImages` | `string[]` | 1-6张 | 产品主图，用于产品列表封面和详情页主图展示 |
| `detailImages` | `string[]` | 0-8张 | 详情图，独立存储，用于产品详情页独立区域展示 |
| `videos` | `string[]` | 0-3个 | 产品视频 |

### 2. 前端产品详情页 - 主图展示区

**位置:** 产品标题下方

**布局:** 左侧缩略图 + 右侧大图

**左侧缩略图:**
- 垂直排列1-6张缩略图
- 点击切换右侧大图
- 超出显示范围时显示上下箭头，点击滚动
- 当前选中缩略图高亮显示

**右侧大图:**
- 显示当前选中缩略图对应的大图
- 支持左右箭头切换上一张/下一张
- 鼠标悬停时显示放大镜效果，右侧/旁边显示局部高清放大区域

### 3. 前端产品详情页 - 详情图展示区

**位置:** 主图展示区下方，独立区域

**布局:** 垂直堆叠

**展示方式:**
- 每张详情图全宽显示
- 图片等比例缩放，宽度100%
- 无文字说明，纯图片展示
- 点击图片可全屏查看，支持左右切换
- 垂直滚动浏览（类似阿里巴巴详情页风格）

### 4. 管理后端 - 产品管理页面

#### 4.1 添加产品页面 (`/products/new`)

三个独立上传区域（侧边栏）：

**产品主图 (1-6张)**
- FileUpload组件
- 最多6张图片
- 第一张自动设为主图
- 白色背景正面图推荐尺寸：1200×1200px

**详情图 (0-8张)**
- FileUpload组件
- 最多8张图片
- 场景图/细节图
- 推荐尺寸：1600×1200px

**产品视频 (0-3个)**
- FileUpload组件
- 最多3个视频
- MP4格式，最大100MB

#### 4.2 编辑产品页面 (`/products/[id]/edit`)

与添加产品页面相同的三个上传区域：
- 显示现有已上传的主图、详情图、视频
- 支持删除和添加
- 保持原有数据结构

#### 4.3 浏览产品页面 (`/products/[id]`)

显示产品详情，包含：
- 产品主图展示
- 详情图展示区域
- 产品视频（如有）
- 所有相关信息

## 技术实现

### API修改

1. **Prisma Schema** - 产品模型添加：
   - `mainImages` - String数组（产品主图URLs）
   - `detailImages` - String数组（详情图URLs）
   - `videos` - String数组（视频URLs）

2. **DTO/Interface 修改:**
   - `CreateProductDto` / `UpdateProductDto` - 添加 `mainImages`, `detailImages`, `videos` 字段
   - 删除原有的 `mainImage` (单图) 和 `images[]` 混合结构

3. **Service 修改:**
   - 更新产品创建/更新逻辑，处理新字段结构
   - 保持向后兼容（可选）

### 前端组件修改

1. **ProductGallery 组件** (`apps/web/components/ProductGallery.tsx`)
   - 重构为左侧缩略图 + 右侧大图布局
   - 添加上下箭头滚动功能
   - 添加左右箭头切换功能
   - 添加鼠标悬停放大镜效果

2. **新增 ProductDetailImages 组件** (`apps/web/components/product/ProductDetailImages.tsx`)
   - 垂直堆叠布局
   - 全宽显示图片
   - 点击全屏查看功能
   - 左右切换支持

3. **ProductDetailContent 修改** (`apps/web/components/ProductDetailContent.tsx`)
   - 替换/更新主图展示组件
   - 添加详情图展示区域
   - 适配新的数据结构

### 管理后端修改

1. **FileUpload 组件** (`apps/admin/components/FileUpload.tsx`)
   - 确保支持多文件上传

2. **添加产品页面** (`apps/admin/app/products/new/page.tsx`)
   - 更新上传区域为三个独立区域
   - 更新提交数据结构

3. **编辑产品页面** (`apps/admin/app/products/[id]/edit/page.tsx`)
   - 同步更新

4. **浏览产品页面** (`apps/admin/app/products/[id]/page.tsx`)
   - 更新图片展示区域
   - 显示主图、详情图、视频
