# CC Scale B2B外贸平台使用手册

## 1. 系统简介

### 1.1 系统概述

CC Scale B2B外贸平台是一套面向衡器（体重秤、吊秤等称重设备）外贸企业的数字化营销管理系统。平台提供完整的产品展示、询盘管理、多语言切换等功能，帮助企业高效开展B2B对外贸易。

**主要功能模块：**

| 模块 | 说明 |
|------|------|
| 产品管理 | 增删改查产品，支持多图、视频、规格参数 |
| 分类管理 | 树状分类结构，支持中英文 |
| 询盘管理 | 客户询价收集、处理、跟进 |
| 用户管理 | 多角色权限控制 |
| 网站设置 | SEO配置、邮件通知等 |

### 1.2 访问地址

| 环境 | 地址 | 说明 |
|------|------|------|
| 前台网站 | `/` | 客户浏览产品、提交询盘 |
| 管理后台 | `/admin` | 管理员操作入口 |

## 2. 前台功能使用

**功能说明：**

- 展示所有已启用的产品列表

| 设备类型 | 列数 | 说明 |
|----------|------|------|
| 桌面端 (≥1280px) | 4列 | 横向排列 |

### 2.2 提交询盘

#### 快速询盘按钮使用

在产品详情页右侧栏，点击 **「立即询盘」** 按钮，弹出快速询盘表单。

**快速询盘表单字段：**

| 字段 | 必填 | 说明 |
|------|------|------|
| 产品 | 自动填充 | 显示当前询价产品 |
| 您的姓名 | 是 | 客户姓名 |
| 电子邮箱 | 是 | 用于回复 |
| 电话号码 | 否 | 联系方式 |
| 公司名称 | 否 | 客户公司 |
| 数量 | 是 | 意向采购数量 |
|留言 | 否 | 补充说明 |

#### 完整询盘表单填写

**填写规范：**

| 字段 | 规范 |
|------|------|
| 姓名 | 真实姓名，中英文均可 |
| 邮箱 | 有效邮箱地址，用于接收回复 |
| 电话 | 国际区号+号码格式，如 +1 2345678900 |
| 公司 | 公司全称 |
| 数量 | 数字，最小为起订量(MOQ) |

提交成功后，页面显示成功提示：

### 2.3 多语言切换

#### 中英文切换方法

**切换位置：** 页面顶部导航栏右侧

**操作方式：** 点击语言切换按钮，在「English」和「中文」之间切换

#### 语言切换后的变化

| 内容项 | 英文版 | 中文版 |
|--------|--------|--------|
| 导航菜单 | 英文 | 中文 |
| 产品名称 | 英文名 | 中文名（如有） |
| 产品描述 | 英文详情 | 中文详情 |
| 按钮文字 | English | 中文 |
| 提示信息 | 英文 | 中文 |

**注意：**

- 部分产品可能没有中文名称，显示英文原名
- 询盘提交后，销售团队可用任一语言回复

## 3. 管理后台使用

### 3.1 登录与权限

#### 登录方法

**访问地址：** `/admin`

**登录凭证：**

| 字段 | 说明 |
|------|------|
| 邮箱 | 管理员账号邮箱 |
| 密码 | 账号密码 |

**登录流程：**
1. 访问 `/admin`
2. 输入邮箱和密码
3. 点击「登录」按钮
4. 登录成功自动跳转管理后台首页

#### 角色权限说明

| 角色 | 代码 | 产品管理 | 分类管理 | 询盘管理 | 用户管理 | 系统设置 |
|------|------|:--------:|:--------:|:--------:|:--------:|:--------:|
| 超级管理员 | super_admin | 全部 | 全部 | 全部 | 全部 | 全部 |
| 管理员 | admin | 全部 | 全部 | 全部 | 全部 | 只读 |
| 编辑 | editor | 创建/编辑 | 创建/编辑 | - | - | - |
| 查看者 | viewer | 只读 | 只读 | 只读 | - | - |

#### 登出操作

**操作路径：** 点击右上角用户头像 → 「退出登录」

### 3.2 产品管理

#### 添加新产品

**访问路径：** `/products/new`

**必填字段：**

| 字段 | 说明 | 示例 |
|------|------|------|
| SKU | 产品唯一标识，不能重复 | `BS-200`, `HS-500` |
| 分类 | 产品所属分类 | Body Scales |
| 产品名称 (英文) | 英文名称 | `Digital Body Scale BS-200` |
| Slug | URL别名 | `digital-body-scale-bs-200` |

**价格与交期：**

| 字段 | 说明 | 示例 |
|------|------|------|
| 最低价 | FOB价格区间最小值 (USD) | `15.00` |
| 最高价 | FOB价格区间最大值 (USD) | `25.00` |
| 最小起订量 | MOQ | `100` |
| 交期 | 生产/交货周期 | `15-20 days` |

**产品图片：**

| 类型 | 数量 | 要求 |
|------|------|------|
| 产品图库 (主图) | 1-5张 | 白色背景正面图，1200×1200px，第一张自动设为主图 |
| 详情图 | 0-8张 | 场景图/细节图，1600×1200px |
| 产品视频 | 0-1个 | MP4格式，≤100MB，时长30-60秒 |

**B2B核心卖点：**

| 字段 | 说明 | 示例 |
|------|------|------|
| 核心卖点 | 产品核心竞争力要点 | `OEM/ODM available - Custom logo` |
| 出口年限 | 出口经验年限 | `3-5 years` |
| 目标市场 | 产品主要销售国家 | `United States`, `Germany` |
| 贸易关键词 | B2B搜索关键词 | `wholesale`, `factory` |

**技术规格：**

| 字段 | 说明 | 示例 |
|------|------|------|
| Label (英文) | 规格名称 | `Capacity` |
| Label (中文) | 规格名称中文 | `最大称重` |
| Value (英文) | 规格值 | `180kg / 400lb` |
| Value (中文) | 规格值中文 | `180公斤 / 400磅` |

**B2B认证（可多选）：**

| 认证 | 说明 | 适用市场 |
|------|------|----------|
| CE | 欧盟安全认证 | 欧洲 |
| FCC | 美国电磁兼容认证 | 美国 |
| RoHS | 欧盟有害物质限制 | 欧洲 |
| ISO9001 | 国际质量管理体系 | 工业产品 |
| UL | 美国安全认证 | 美国/加拿大 |
| FDA | 食品和药物管理局认证 | 食品/医疗 |

**SEO设置：**

| 字段 | 最大长度 | 说明 |
|------|----------|------|
| SEO标题 (英文) | 60字符 | 搜索引擎优化标题 |
| SEO标题 (中文) | 30字符 | 中文SEO标题 |
| SEO描述 (英文) | 160字符 | Meta描述 |
| SEO描述 (中文) | 80字符 | 中文Meta描述 |
| SEO关键词 (英文) | 50字符 | 逗号分隔 |
| SEO关键词 (中文) | 30字符 | 逗号分隔 |

**状态设置：**

| 字段 | 说明 |
|------|------|
| 启用 | 是=前台显示，否=隐藏 |
| 精选产品 | 是=显示在推荐区域 |
| 排序 | 数字越小越靠前 |

**保存流程：**
1. 填写完所有信息
2. 点击右下角「保存产品」
3. 显示「保存成功」提示
4. 自动跳转产品列表页

#### 编辑产品信息

**访问路径：** `/products/[id]/edit`

**功能说明：**
- 已有信息自动填充到表单
- 已有图片显示缩略图预览
- 可删除已有图片（点击X按钮）
- 可重新上传新图片替换
- 保存后自动更新

#### 管理产品图片

**上传规范：**

| 类型 | 尺寸 | 格式 | 大小限制 |
|------|------|------|----------|
| 主图 | 1200×1200px | JPG/WebP | ≤500KB/张 |
| 详情图 | 1600×1200px | JPG/WebP | ≤800KB/张 |
| 视频 | 1920×1080px | MP4 | ≤100MB |

**图片质量要求：**
- 白色或浅色背景
- 产品居中，占图片80%以上
- 光线均匀，无阴影
- 无水印、无logo（除非OEM）

#### 删除产品

**操作方法：**
1. 进入产品列表页 `/products`
2. 勾选要删除的产品（可多选）
3. 点击「删除」按钮
4. 确认删除操作

**注意：** 删除后数据无法恢复，请谨慎操作。

### 3.4 询盘管理

#### 查看询盘列表

**访问路径：** `/inquiries`

**列表字段说明：**

| 字段 | 说明 |
|------|------|
| 询盘ID | 自动生成的唯一标识 |
| 客户姓名 | 客户填写的姓名 |
| 邮箱 | 客户邮箱 |
| 电话 | 客户电话 |
| 公司 | 客户公司名称 |
| 产品 | 询价的产品名称 |
| 数量 | 客户想要的数量 |
| 留言 | 客户留言信息 |
| 状态 | 询盘处理状态 |
| 提交时间 | 询盘提交时间 |

**询盘状态说明：**

| 状态 | 代码 | 说明 |
|------|------|------|
| 待处理 | pending | 新收到的询价，需要销售跟进 |
| 已处理 | processed | 已确认/已回复，等待进一步沟通 |
| 已回复 | replied | 已发送报价或回复给客户 |
| 已成交 | closed | 询价已转化为订单 |
| 无效 | invalid | 无效询价（客户无意向等） |

#### 处理询盘状态

| 操作 | 说明 |
|------|------|
| 查看详情 | 点击查看完整询价信息 |
| 标记已处理 | 将状态改为「已处理」 |
| 回复客户 | 通过邮件回复客户 |

**状态更新流程：**
1. 点击询盘查看详情
2. 根据跟进情况更新状态
3. 可通过邮件功能回复客户

#### 回复询盘

**操作方法：**
1. 点击询盘右侧「回复」按钮
2. 在邮件表单中填写回复内容
3. 点击发送

#### 导出询盘数据

**操作方法：**
1. 在询盘列表页勾选要导出的询盘
2. 点击「导出」按钮
3. 下载CSV/Excel格式文件

## 4. 常见问题

### 4.1 登录问题

**Q: 忘记密码怎么办？**
A: 联系超级管理员重置密码。

**Q: 账号被锁定？**
A: 检查账号状态是否被禁用，联系管理员启用。

**Q: 登录后显示无权限？**
A: 检查当前账号角色是否有该功能访问权限。

### 4.2 图片上传问题

**Q: 产品图片上传失败？**
A: 可能原因及解决方法：
- 文件太大：压缩图片到500KB以下
- 格式不支持：使用JPG或WebP格式
- 网络中断：检查网络连接后重试

**Q: 详情图不显示？**
A: 解决方法：
- 重新上传图片
- 确保保存成功
- 清除浏览器缓存

**Q: 视频无法播放？**
A: 可能原因：
- 格式不支持：转换为MP4格式
- 文件损坏：重新上传
- 大小超限：确保≤100MB

### 4.3 询盘未收到问题

**Q: 客户提交询盘但未收到邮件通知？**
A: 检查项：
1. 邮件通知设置是否正确配置
2. 邮件可能被拦截，检查垃圾邮件文件夹
3. 管理员邮箱是否正确

**Q: 询盘状态含义？**
A: 请参考3.4节的询盘状态说明。
