## 1. 产品管理

### 1.1 添加产品

访问路径: `/products/new`

#### 基本信息

| 字段 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| SKU | 文本 | ✅ | 产品唯一标识符，不能重复 | `BS-200`, `HS-500` |
| 分类 | 下拉 | ✅ | 选择产品所属分类 | Body Scales, Hanging Scales |
| 产品名称 (英文) | 文本 | ✅ | 英文名称，用于国际化显示 | `Digital Body Scale BS-200` |
| 产品名称 (中文) | 文本 | ❌ | 中文名称 | `数字体重秤 BS-200` |
| URL别名 | 文本 | ❌ | SEO友好URL，默认为名称拼音 | `digital-body-scale-bs-200` |
| 简短描述 (英文) | 文本 | ❌ | 产品一句话简介 | `High-precision digital body scale` |
| 简短描述 (中文) | 文本 | ❌ | 中文简介 | `高精度数字体重秤` |
| 详细描述 (英文) | 多行文本 | ❌ | 产品详细介绍 | 富文本内容 |
| 详细描述 (中文) | 多行文本 | ❌ | 中文详细介绍 | 富文本内容 |

#### 价格与库存

| 字段 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| 最低价格 | 数字 | ❌ | FOB价格区间最小值 (USD) | `15.00` |
| 最高价格 | 数字 | ❌ | FOB价格区间最大值 (USD) | `25.00` |
| 最小起订量 (MOQ) | 数字 | ❌ | Minimum Order Quantity | `100` |
| 交期 | 文本 | ❌ | 生产/交货周期 | `15-20 days`, `25-35 days` |

#### 产品图库

| 字段 | 类型 | 必填 | 要求 | 说明 |
|------|------|------|------|------|
| 产品图库 (1-5张) | 图片 | ✅ | 白色背景正面图，白底 | 第一张自动设为主图 |
| 详情图 (0-8张) | 图片 | ❌ | 场景图/细节图 | 展示产品细节和应用场景 |
| 产品视频 | 视频 | ❌ | MP4/WebM, ≤200MB | 产品演示视频 |

**详细规格见 [5. 上传规格](#5-上传规格)**

#### 技术规格

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 规格项 | 键值对 | ❌ | 可添加多条规格，如Capacity, Division等 |
| Label (英文) | 文本 | ✅ | 规格名称英文 | `Capacity`, `Division` |
| Label (中文) | 文本 | ❌ | 规格名称中文 | `最大称重`, `分度值` |
| Value (英文) | 文本 | ✅ | 规格值英文 | `180kg / 400lb` |
| Value (中文) | 文本 | ❌ | 规格值中文 | `180公斤 / 400磅` |

#### B2B核心卖点

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 核心卖点 | 列表 | ❌ | 列出产品的核心竞争力要点 |

示例:
- `OEM/ODM available - Custom logo and packaging`
- `ISO9001 certified factory with 20+ years experience`
- `Fast production lead time: 15-25 days`

#### B2B应用场景

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| Application Scenarios (英文) | 多行文本 | ❌ | 描述产品使用场景 |
| 应用场景 (中文) | 多行文本 | ❌ | 中文应用场景描述 |

#### B2B认证

**可多选**，存储为JSON数组格式。

| 选项 | 代码 | 说明 |
|------|------|------|
| CE | CE | **欧盟安全认证** - 表示产品符合欧盟安全、健康、环保要求，适用于欧盟市场 |
| FCC | FCC | **美国联邦通信委员会认证** - 表示产品符合美国电磁兼容要求，适用于美国市场 |
| RoHS | RoHS | **欧盟有害物质限制认证** - 表示产品不含有害物质，符合环保要求，适用于欧盟市场 |
| ISO9001 | ISO9001 | **国际质量管理体系认证** - 表示工厂已通过ISO质量管理体系认证，质量管理规范 |
| UL | UL | **美国安全认证** - 表示产品通过美国安全实验室认证，适用于美国/加拿大市场 |
| FDA | FDA | **美国食品和药物管理局认证** - 表示产品符合FDA要求，适用于食品/医疗相关产品 |

**选择建议**:
- 销往欧洲: 必选 CE, RoHS
- 销往美国: 必选 FCC, UL (如适用)
- 工业产品: 建议 ISO9001
- 食品/医疗: 必选 FDA

#### B2B FAQ

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| Question (英文) | 文本 | ✅ | 常见问题英文 |
| Question (中文) | 文本 | ❌ | 常见问题中文 |
| Answer (英文) | 文本 | ✅ | 问题答案英文 |
| Answer (中文) | 文本 | ❌ | 问题答案中文 |

#### 贸易信息

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| HS编码 | 文本 | 海关商品编码 | `8423.82` |
| 付款条款 | 文本 | Payment Terms | 见下方选项说明 |
| 贸易条款 | 文本 | Shipping Terms | 见下方选项说明 |
| 保修期 | 文本 | Warranty Information | `1 year warranty` |

**付款条款选项:**

| 选项 | 说明 |
|------|------|
| T/T 30/70 | **电汇30%定金+70%发货前结清** - 最常用，适合首次合作 |
| T/T 50/50 | **电汇50%定金+50%发货前结清** - 平衡风险，双方信任度高 |
| T/T 100% | **电汇100%预付** - 要求最高，适合小额订单或信任客户 |
| L/C | **信用证付款** - 银行担保，适合大额订单，安全性高 |
| L/C at sight | **即期信用证** - 开证行见单即付，风险最低，费用较高 |
| PayPal | **贝宝支付** - 在线支付，方便快捷，有手续费（~4%） |
| West Union | **西联汇款** - 传统汇款方式，适用于小额或个人交易 |
| Trade Assurance | **阿里巴巴信保** - 平台担保交易，安全性高 |

**贸易条款选项:**

| 选项 | 说明 |
|------|------|
| FOB Shanghai | **离岸价-上海** - 卖方负责将货物送上船，费用到上海港止，买方承担运费和保险 |
| FOB Ningbo | **离岸价-宁波** - 卖方负责将货物送上船，费用到宁波港止，买方承担运费和保险 |
| FOB Shenzhen | **离岸价-深圳** - 卖方负责将货物送上船，费用到深圳港止，买方承担运费和保险 |
| CIF [Port] | **到岸价** - 卖方承担运费和保险到指定目的港，如CIF Los Angeles |
| CIF [Port] | **成本+运费+保险** - 卖方负责运输和保险直到目的港 |
| EXW [City] | **工厂交货** - 买方到卖方工厂提货，费用和风险最低 |
| DDP | **完税后交货** - 卖方承担所有费用和风险直到货物到达买方指定地点 |
| FOB [Port] | **自定义港口** - 可指定其他起运港 |

#### 包装信息

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| Packaging Info (英文) | 多行文本 | 英文包装规格 | `10 pcs/carton, 42x38x28 cm, G.W. 12kg` |
| 包装信息 (中文) | 多行文本 | 中文包装规格 | `10个/箱，42x38x28厘米，毛重12公斤` |

| 毛重 | 一箱的重量（含包装） | `G.W. 12kg` |
| 净重 | 产品本身的重量（不含包装） | `N.W. 10kg` |
| 立方数 | 体积立方米数（用于计算运费） | `0.045 CBM` |

#### SEO优化

| 字段 | 类型 | 说明 | 最大长度 |

#### 状态设置

## 3. 询价管理

### 3.1 询价列表

访问路径: `/inquiries`

显示所有客户提交的询价请求，包含：

| 字段 | 说明 |
|------|------|
| 询价ID | 自动生成的唯一标识 |
| 客户姓名 | 客户填写的姓名 |
| 邮箱 | 客户邮箱 |
| 电话 | 客户电话 |
| 公司 | 客户公司名称 |
| 产品 | 询价的产品名称 |
| 数量 | 客户想要的数量 |
| 留言 | 客户留言信息 |
| 状态 | 见下方选项说明 |
| 提交时间 | 询价提交时间 |

**询价状态选项:**

| 状态 | 代码 | 说明 |
|------|------|------|
| 待处理 | pending | **新收到的询价** - 需要销售团队跟进处理 |
| 已处理 | processed | **已确认/已回复** - 已联系客户，等待进一步沟通 |
| 已回复 | replied | **已发送报价** - 已发送正式报价或回复给客户 |
| 已成交 | closed | **询价已转化为订单** - 客户已下单 |
| 无效 | invalid | **无效询价** - 如客户无意向、信息虚假等 |

### 3.2 处理询价

| 操作 | 说明 |
|------|------|
| 查看详情 | 点击查看完整询价信息 |
| 标记已处理 | 将状态改为"已处理" |
| 回复客户 | 通过邮件回复客户 |

## 5. 上传规格

### 5.1 产品图片

#### 产品图库 (主图)

| 参数 | 要求 |
|------|------|
| 数量 | 1-5张 |
| 尺寸 | 1200×1200px 或 2000×2000px |
| 背景 | 白色背景，正面图 |
| 格式 | JPG/WebP |
| 大小 | ≤5MB/张（自动压缩优化） |
| 用途 | 产品列表缩略图、搜索结果图 |

**建议**:
- 使用白色无缝背景
- 产品居中，占图片80%以上
- 光线均匀，无阴影
- 高分辨率，便于放大查看

#### 详情图

| 参数 | 要求 |
|------|------|
| 数量 | 0-8张 |
| 尺寸 | 1600×900px (16:9) 或 1200×800px |
| 背景 | 白色或浅色背景 |
| 格式 | JPG/WebP |
| 大小 | ≤5MB/张（自动压缩优化） |
| 用途 | 展示产品细节、应用场景、尺寸图 |

**建议**:
- 包含产品正面、背面、侧面特写
- 包含使用场景图
- 包含尺寸/规格示意图
- 高清但文件不宜过大

#### 缩略图（前端自动生成）

| 位置 | 显示尺寸 |
|------|---------|
| 左侧缩略图栏 | 80×80px (Desktop), 64×64px (Mobile) |
| 产品列表 | 150×112px (4:3) |
| 相关产品 | 150×112px (4:3) |

### 5.2 产品视频

| 参数 | 要求 |
|------|------|
| 数量 | 0-1个 |
| 格式 | MP4, WebM |
| 分辨率 | 1920×1080px (1080p) |
| 时长 | 30秒 - 1分钟 |
| 大小 | ≤200MB |
| 用途 | 产品演示、功能展示 |

**建议**:
- 展示产品外观和功能
- 包含产品特点和应用场景
- 配乐或旁白（可选）
- 专业灯光和拍摄

### 5.3 其他上传

| 类型 | 尺寸要求 | 格式 | 大小限制 |
|------|---------|------|----------|
| 客户评价头像 | 200×200px | JPG/PNG | ≤3MB |
| 客户标志 | 200×80px (透明PNG) | PNG | ≤2MB |

### 7.1 产品接口

#### 获取产品列表

- 格式：最多2位小数

### 起订量 (MOQ)

- 范围：1 - 999999
- 类型：整数

### 交期

- 长度：1-100字符
- 示例格式：`15-20 days`, `25-35 days`

## 附录B：常见问题

### Q1: 产品图片上传失败？
**原因**: 文件太大或格式不支持
**解决**: 压缩图片到≤5MB，使用JPG格式

### Q2: SKU重复？
**原因**: 已存在相同SKU的产品
**解决**: 使用不同的SKU，如添加后缀 `BS-200A`

### Q3: 详情图不显示？
**原因**: 图片URL失效或未正确保存
**解决**: 重新上传图片，确保保存成功

### Q4: 视频无法播放？
**原因**: 格式不支持或文件损坏
**解决**: 转换为MP4格式，确保≤200MB

### Q5: 产品不显示在前台？
**检查项**:
1. 是否已启用（isActive=true）
2. 是否已分配分类
3. 是否有主图

## 附录C：最佳实践

### 产品图片拍摄指南

1. **光线**: 使用柔和的自然光或专业摄影灯
2. **背景**: 纯白色或浅灰色背景
3. **角度**: 正面45度角，展示产品全貌
4. **清晰度**: 确保产品细节清晰可见
5. **一致性**: 所有产品使用相同拍摄风格

### 产品描述写作

1. **标题**: 简洁，包含关键产品名和型号
2. **描述**: 包含用途、特点、规格、应用场景
3. **关键词**: 包含采购商可能搜索的词汇
4. **差异化**: 突出与竞争对手的差异

### SEO优化建议

1. **标题**: 包含产品名 + 品牌 + 型号
2. **描述**: 包含核心卖点 + 应用场景
3. **关键词**: 包含产品词 + 行业词 + 采购词
